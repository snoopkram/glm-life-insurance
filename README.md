# glm-life-insurance
# Pricing einer lebenslangen Rente mittels Sterblichkeits-GLMs

## Über das Projekt
Dieses Projekt analysiert historische Sterblichkeitstrends und modelliert zukünftige Sterbenswahrscheinlichkeiten mithilfe von Generalized Linear Models (GLMs). Ziel ist es, die Auswirkungen der steigenden Lebenserwartung (Langlebigkeitsrisiko) auf das Pricing von Rentenversicherungen zu quantifizieren.

## Datenquelle
* Human Mortality Database (HMD) - Sterbetafeln und Populationsdaten.

## Verwendete Tools
* [R / Python] für Datenaufbereitung und Modellierung
* GLM (Poisson-Verteilung)
* Finanzmathematik (Barwertberechnung)
